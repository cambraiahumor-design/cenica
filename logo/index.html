Copie o código SVG principal para colar no Figma, Canva ou HTML.

Como usar no Instagram e LinkedIn

  1. Baixe o SVG da versão escura (principal)
  2. Abra no navegador → clique com botão direito → "Salvar imagem"
  3. Ou importe no Canva: Criar design → Quadrado (1080x1080) → Importar SVG
  4. No Canva, exporte como PNG 1080x1080 (qualidade máxima)
  5. Use o PNG como foto de perfil em todas as redes

💡 Dica: O Instagram corta a foto de perfil em círculo automaticamente. O mark já foi desenhado para funcionar bem nesse formato.

Regras de Uso

Proteção da marca

Essas regras garantem que a logo seja sempre reconhecível e nunca perca impacto.

✓ Fazer

Usar o logo mark como foto de perfil em todas as redes sociais com fundo escuro.

✓ Fazer

Manter espaço de respiro ao redor do logo equivalente à altura da letra C.

✓ Fazer

Usar sempre as cores oficiais: Spotlight (#D4863A) ou Cream (#F5EFE6) para o C.

✗ Não fazer

Nunca distorcer, rotacionar ou alterar as proporções do logo.

✗ Não fazer

Nunca usar o logo sobre fundos que criem baixo contraste com o C.

✗ Não fazer

Nunca adicionar sombras, efeitos ou filtros externos ao logo original.
